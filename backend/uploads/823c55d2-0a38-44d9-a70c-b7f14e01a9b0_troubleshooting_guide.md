# Troubleshooting Guide

## Common Issues

### Cannot Log In
- Check caps lock is off
- Clear browser cache and cookies
- Try incognito/private mode
- Reset password via forgot-password link
- If issue persists, check if account is locked (3+ failed attempts locks for 1 hour)

### Order Not Received
- Verify shipping address in order confirmation email
- Check carrier tracking link (provided in dispatch email)
- Allow 2 extra business days for delays
- If 5+ days overdue, open a support ticket with order ID

### Payment Declined
- Confirm billing address matches bank records
- Check card expiry date
- Try a different payment method
- Contact bank to rule out fraud block

## Escalation Triggers
Immediately escalate to L2 if:
- Data breach suspected
- Service completely unavailable for 2+ hours
- Customer threatens legal action
- Media or PR inquiry received
